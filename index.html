About

Neutral colors brings a beautiful color scheme to your terminal that will improve your command line experience. There are two different versions: Dark and Light, both can be used on OS X, Windows and Linux. You can find a preview of the themes below.

Install

Depending on your operating system and your terminal emulator the theme installation could be slightly different, choose the platform you're using.
Windows Terminal
Copy content from./theme/windows-terminal/Neutral*.json into your config file
Tilix
Copy file ./theme/tilix/Neutral*.json in ~/.config/tilix/schemes
Konsole
Copy file ./theme/konsole/Neutral*.colorscheme in ~/.local/share/konsole/
Termux
Copy file content from one file./theme/termux/*.properties in ~/.termux/colors.properties
iTerm2
Import file ./theme/iterm/*.properties
VScode
Copy file content from one file./theme/vscode/*.properties VScode config
